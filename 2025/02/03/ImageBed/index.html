使用 GitHub + PicGo + jsDelivr 搭建免费图床

1. 准备工作

GitHub 账号：确保你有一个 GitHub 账号。

安装 PicGo：下载并安装 PicGo（支持 Windows、macOS、Linux）。
https://github.com/Molunerfinn/PicGo/releases

2.创建 GitHub 仓库

  1. 新建仓库
    登录 GitHub，点击右上角 + → New repository。
    输入仓库名（如 image-bed），选择 Public（私有仓库无法通过 jsDelivr 访问）。
    勾选 Add a README file（可选），点击 Create repository。
  2. 生成 Token
    点击 GitHub 右上角头像 → Settings → Developer settings → Personal access tokens → Tokens (classic)。
    点击 Generate new token → 输入名称（如 picgo-token），勾选 repo 权限。
    生成后复制 Token（只显示一次，务必保存好）。

3.配置 PicGo

  1. 打开 PicGo
    打开安装好的 PicGo，进入 图床设置 → GitHub图床。
  2. 写配置信息
    仓库名：你的GitHub用户名/仓库名（例如 jeseep1nkman/image-bed）。
    分支名：默认 main 或 master（根据仓库实际分支填写）。
    Token：粘贴刚刚生成的 GitHub Token。
    存储路径：可选，例如 images/（图片会存储在仓库的 images 目录下）。

高速：jsDelivr 提供全球 CDN 加速。
便捷：PicGo 支持拖拽上传、剪贴板上传，一键生成链接。
